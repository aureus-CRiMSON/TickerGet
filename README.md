# TickerGet
This tool can be used to read the memory of LDJ to get the segmented display's text (without the scrolling effect).

As of now, it does not do much other than print said data out to a log file located in the root directory in the game.

## Supported Versions
2025091710-010

## Goals
* Serial Port Communication
* Network Communication
* Get Song Difficulty
* Get Song Artist
* Get 1p/2p information on each song as well
* Get Song Category (game version)
* Get Song Genre
